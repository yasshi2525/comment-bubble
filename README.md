# コメントを打たれる撃たれるとツール (フリー版)

* https://namagame.coe.nicovideo.jp/games/lg12187 のソースコードです
* ライセンスのため一部の画像と音声データが公開版と異なります。（無音データが格納されています。）

# インストール方法・動作確認方法

```sh
npm install
npm start
```
